Nuri Infos application framework, NuriAF, is based on the Spring Framework and applies the latest open-source frameworks to provide the most optimal technology base for developing enterprise web applications

NuriAF provides basic functions for enterprise web applications, such as user management, permission management, and bulletin board management, allowing you to focus more on the system's business functions and reducing development costs

· Utilizes common components provided by the framework and customizes existing common components.
