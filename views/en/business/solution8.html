Nuri Infos' AutoPlatform is a 3-Tier C/S source generation technology solution designed to improve productivity and processing speed.

This solution leverages 3-Tier Client/Server source generation technology to automatically create server and client source code. Efficient interoperation with a communication module, which acts as a middleware replacement, enables load balancing of business transactions, leading to gains in productivity and processing speed.

Service
01 Reduced Development Time
· Enhanced Productivity and Shorter Project Timelines: Achieved through the application of automated processes.
· Automatic Generation of Service and Middleware-Related Source Code and Components: Streamlines development.
· Automatic Generation of Variable Names and Basic SQL Queries: Reduces manual coding and potential errors.
02Fast Processing Speed
· Guaranteed High Performance: Leverages a lightweight client-server architecture to exceed existing processing speeds.
· Proprietary Core Gate (Server): A custom-developed server core for optimized performance.
· Delphi 4GL Tool (Client): Utilizes the Delphi 4GL tool for client-side development.
03Cost-Effectiveness
· Reduced Development Costs: Achieved through automated processes and shorter development cycles.
· Independent of Client PC Performance: Runs efficiently even on low-specification PCs.
· Runs on Low-Spec PCs: Compatible with Pentium-class PCs with 64MB RAM.
· Increased TP-Monitor Utilization: Supports the same number of concurrent users with enhanced efficiency.
04Ease of Maintenance
· Server Monitoring Tools Provided: Includes tools for monitoring Core Gate and KQ-related processes.
· Easy Service Modification and New Development: Simplified by prior developer training.
