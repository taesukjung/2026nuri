누리인포스의 애플리케이션 프레임워크인 NuriAF 은 Spring Framework 기반으로 최신의 오픈소스
프레임워크를 적용하여 기업용 웹 애플리케이션을 개발하는데 가장 최적의 기술 기반을 제공합니다.

기업용 웹 애플리케이션에 기본적인 기능인 사용자 관리/권한관리/게시판관리 기능을 기본 제공하여 시스템의 비즈니스 기능에 더욱 집중하도록 하며 개발비 절감효과가 있습니다.

Service
01최신 버전 오픈소스 프레임워크 적용
Spring Framework 4 – Base Framework
Spring Security 4 – 인증 및 보안
myBatis – 트랜잭션 처리
HTML5 UI Component(상용) – HTML5 지원 UI 개발
02외부 시스템 인터페이스 표준화를 위한 Open API 서비스
RESTFul 기반 Open API 서비스를 통해 향후 Client POS 를 포함한 외부 시스템 인터페이스 방식의 표준화 및 안정성 지원
03Cross Browser 및 웹 호환성 확보
HTML5 UI Component 적용으로 HTML5 지원 및 Browser 버전업에 따른 호환성 확보
UI 개발 시 HTML 코드를 사용하지 않고 HTML5 UI Component에서 제공하는 Method 만으로 UI 디자인 및 Logic 구성
Browser 렌더링 HTML 코드는 HTML5 UI Component에서 자동 생성
04개발 및 운영 편의성
단순한 공통 모듈 라이브러리 수준이 아닌 아키텍처, 개발 툴, 디자인 패턴 등을 담고 있는 어플리케이션 개발 / 운영 플랫폼 제공
비즈니스 모듈 간 상호종속성 최소화
응용S/W 레이어 설계로 공통모듈의 재 사용성 향상
프레임워크에서 제공하는 공통 컴포넌트 활용 및 기존 공통컴포넌트 커스트마이징
